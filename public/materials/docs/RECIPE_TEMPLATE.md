# Material Recipe Template

Use this template when creating a new realtime material or VFX material page.

## Visual Goal

- What should the material look like?
- What is the strongest visual read?
- What should the viewer notice first?

## Production Use Case

- Where is this material used?
- Is it a hero effect, background effect, or cheap repeated effect?

## Core Components

- UV setup
- Mask logic
- Color gradient
- Emission
- Alpha / opacity
- Optional distortion

## Node Graph Steps

1. Input data
2. UV setup
3. Mask creation
4. Motion / animation
5. Color assembly
6. Emission
7. Alpha / opacity
8. Final output

## Artist Controls

Color, speed, threshold, softness, intensity, variation.

## Optimization Notes

Track texture samples, expensive nodes, low-end fallbacks, and LOD or distance fallbacks.
