# Image Recipe Pilot

This document defines the image-generation standard for material-library visual anchors.

## Output Standard

- Aspect ratio: 16:9
- Target resolution: 1920x1080
- Composition: full-bleed widescreen hero
- Text in image: none
- Watermark: none

## Acceptance Check

- The image works in a 16:9 card without letterboxing.
- The material behavior is readable within two seconds.
- The subject is not trapped inside a square poster composition.
