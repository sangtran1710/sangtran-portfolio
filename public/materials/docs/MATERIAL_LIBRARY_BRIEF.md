# UE5 Material Library Brief

This library is a production-oriented material reference, not a random list of shader experiments.
It is stylized-first, but not anime-only or cel-shader-only.

## Purpose

Separate reusable logic, style direction, buildable recipes, and production notes so the library can scale without becoming hard to navigate.
The style scope includes stylized, anime, cel shader, semi-realistic, and realistic material work.

## Main Lanes

- Core Packs: reusable material logic.
- Style Direction: lookdev language for stylized, anime, cel shader, semi-realistic, and realistic material families.
- Recipes: buildable material effects.
- Production Notes: issue-driven production support.

## Authoring Principles

Keep each page easy to scan, expose useful artist controls, document limitations, and extract repeated patterns into core packs.

## Style Bias

The library should favor stylized production thinking first: readable shapes, authored control, clear material instances, and intentional tradeoffs. Realistic and semi-realistic pages still belong here, but they should follow the same production structure instead of becoming separate loose notes.
